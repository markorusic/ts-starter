# parcel-typescript-example

TypeScript starter pack with [Parcel Bundler](https://parceljs.org/) and eslint + prettier.

## Usage

### Installation

```sh
$ npm i
```

### Development

```sh
$ npm run dev
```
    
### Build

```sh
$ npm run build
```
